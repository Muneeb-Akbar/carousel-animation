🎠 Simple Carousel Slider
A lightweight, animated image slider built with HTML, CSS, and JavaScript. This minimal yet stylish carousel provides smooth transitions and is easy to customize.


🚀 Features
✅ Automatic & manual slide navigation
✅ Smooth CSS animations for seamless transitions
✅ Fully responsive design for all devices
✅ Minimal JavaScript for better performance
✅ Easy customization with simple HTML & CSS

⚡ Usage
🔹 Add Your Own Images
Place your images inside the images/ folder and update the HTML accordingly.

🎨 Customize Styles
Modify style.css to change colors, font sizes, or transitions.

⚙️ Modify JavaScript
Update script.js to change animation speed, autoplay, or navigation behavior.

🎨 Customization
🔹 Change Slide Duration: Edit the setInterval() function in script.js.
🔹 Modify Transition Effects: Update the transition properties in style.css.
🔹 Add Navigation Arrows: Customize the prev and next buttons.
